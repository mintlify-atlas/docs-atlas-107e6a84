ActivityWatch is a free, open-source, privacy-first time tracker that runs entirely on your machine. It automatically records your active application, window title, browser tab, and AFK (away-from-keyboard) status — giving you a complete picture of how you spend your time without sending any data to external servers.

## How ActivityWatch works

ActivityWatch consists of a local server, a set of watchers that collect data, and a web UI for exploring your activity. All data is stored on your machine.

<Steps>
  <Step title="Install ActivityWatch">
    Download the latest release for your platform (Windows, macOS, or Linux) and run the installer. ActivityWatch starts a local server on port 5600.
  </Step>
  <Step title="Watchers collect your activity">
    Built-in watchers automatically record the active window and application, AFK/idle status, and — with the browser extension — the active browser tab and URL.
  </Step>
  <Step title="Explore your data">
    Open the web dashboard at `http://localhost:5600` to see activity timelines, summaries, and category breakdowns for any time range.
  </Step>
  <Step title="Extend and automate">
    Use the REST API or Python client library to build custom watchers, export data, or integrate ActivityWatch with other tools.
  </Step>
</Steps>
